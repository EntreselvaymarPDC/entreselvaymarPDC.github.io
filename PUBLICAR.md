# Publicar gratuitamente con GitHub Pages

Para obtener exactamente `https://entreselvaymar.github.io`, la cuenta u organización de GitHub debe llamarse `entreselvaymar` y el repositorio público debe llamarse `entreselvaymar.github.io`.

1. Crea el repositorio público `entreselvaymar.github.io`.
2. Autoriza ese repositorio en la conexión de GitHub de Codex.
3. Pide a Codex: "Sube el paquete de GitHub Pages y activa la publicación".

Si tu usuario de GitHub tiene otro nombre, la dirección gratuita será `https://USUARIO.github.io/NOMBRE-DEL-REPOSITORIO/`.
